# Kicks eCommerce Example 

## How to run the project 

To run the the frontend solution `npm i && npm run dev`.

To run the sanity local version `npm i && sanity start`.

To run storybook `npm run storybook`.

### Frontend
- React (Gatsby)
- Storybook 
- Sanity CMS
